# 🚀 Shamil Ahammed T | Professional Portfolio

A high-performance, ultra-responsive portfolio website built with **Astro** and **Tailwind CSS**. Designed for speed, accessibility, and a premium user experience.

## ✨ Key Features

- **⚡ Perfect Performance**: Optimized to achieve 100/100 scores across all Lighthouse categories (Performance, Accessibility, Best Practices, SEO).
- **🎨 Modern Design**: Sleek dark-mode aesthetic with glassmorphism effects and custom cyan gradients.
- **🌀 Interactive Elements**: 
  - **Custom Cursor**: A smooth, GPU-accelerated trailing cursor.
  - **Rhythmic Background**: Dynamic, rhythmic background ripples that respond to clicks and appear randomly.
  - **Smooth Animations**: Intersection observer-based scroll animations for a dynamic feel.
- **📱 Ultra Responsive**: Fluid layout that looks professional on everything from small mobile devices to large desktop monitors.
- **🖼️ Image Optimization**: Leveraging Astro's Image component for next-gen formats (WebP/AVIF) and responsive sizing.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.17.1 or higher)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/shamilahmdt/shamilahmdt.github.io.git
   ```
2. Navigate to the project directory:
   ```bash
   cd shamilahmdt.github.io
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the local development server:
```bash
npm run dev
```

### Production Build

To verify the perfect Lighthouse scores, run a production build:
```bash
npm run build
npm run preview
```
