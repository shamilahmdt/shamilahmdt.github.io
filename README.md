## ✨ Overview

A modern, high-performance portfolio website built to showcase my work, skills, and projects as a Full Stack Developer. Designed with a strong emphasis on **performance, accessibility, and smooth user experience**.

---

## ⚡ Key Features

* 🚀 **Optimized Performance**
  Built for speed with lightweight architecture and efficient rendering.

* 🎨 **Modern UI Design**
  Dark-themed interface with glassmorphism effects and gradient accents.

* 🌀 **Interactive Experience**

  * Custom animated cursor with smooth trailing effect
  * Dynamic background ripple animations
  * Scroll-based reveal animations

* 📱 **Responsive Design**
  Fully optimized for mobile, tablet, and desktop screens.

* 🖼️ **Advanced Image Handling**
  Uses modern formats (WebP/AVIF) for faster loading and better quality.

## 🚀 Getting Started

### Prerequisites

* Node.js (v18+)
* npm

---

### Installation

```bash id="c1"
git clone https://github.com/shamilahmdt/shamilahmdt.github.io.git
cd shamilahmdt.github.io
npm install
```

---

### Run Locally

```bash id="c2"
npm run dev
```

---

### Build for Production

```bash id="c3"
npm run build
npm run preview
```
